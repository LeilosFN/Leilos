# LeilosFN / Leilos

Este repositorio contiene el código fuente de la **WEB Oficial del proyecto LeilosFN**.  
Su función es centralizar toda la información pública del proyecto y ofrecer un punto de acceso claro y actualizado para los usuarios.

## URL Oficial

La página web puede visitarse en:

**https://leilos.qzz.io**

Desde esta dirección los usuarios pueden acceder a:

- Información general del proyecto  
- Noticias y actualizaciones  
- Enlaces a otros repositorios relacionados  
- Recursos y documentación  
- Acceso a herramientas y servicios del ecosistema LeilosFN  

## Estado del proyecto

El sitio web se encuentra en desarrollo activo.  
Es posible que algunas secciones cambien o se actualicen con frecuencia mientras se implementan nuevas funciones y mejoras.

## Objetivo del repositorio

- Mantener el código de la web oficial  
- Facilitar contribuciones y reportes de errores  
- Servir como referencia para la estructura y evolución del proyecto
